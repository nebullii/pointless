# Roadmap

## Phase 1: MVP
- Webcam capture
- Hand landmarks via MediaPipe
- Rule-based gestures: swipe left/right, open palm, fist, pinch
- Command dispatch to presentation apps

## Phase 2: Meeting Integrations
- Zoom and Google Meet shortcuts
- Active window detection + safe routing

## Phase 3: Productization
- Settings UI
- Calibration wizard
- Packaging
