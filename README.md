# Pointless

Hand-gesture control for presentations and meeting apps.

## Status
Skeleton project structure for the MVP.

## Quick Start (dev)
1. Create a venv
2. Install deps (to be added)
3. Run: `python -m app.main`

## Structure
- `app/` entrypoints
- `core/` config and shared utilities
- `vision/` camera capture + hand tracking
- `gestures/` gesture engine and rules
- `actions/` command mapping + dispatch
- `profiles/` app shortcut profiles
- `ui/` overlay + calibration
- `tests/` basic tests
- `docs/` architecture and roadmap
